# Mobile App Assets

Place the following image assets in this folder:

## Required Images

1. **icon.png** (1024x1024) - App icon for iOS and Android
2. **splash.png** (1284x2778) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
4. **favicon.png** (48x48) - Web favicon

## Recommended Specifications

### App Icon (icon.png)

- Size: 1024x1024 pixels
- Format: PNG (no transparency for iOS)
- Background: Solid color (#4F46E5 primary)

### Splash Screen (splash.png)

- Size: 1284x2778 pixels (iPhone 14 Pro Max)
- Format: PNG
- Content: Logo centered, background #4F46E5

### Adaptive Icon (adaptive-icon.png)

- Size: 1024x1024 pixels
- Format: PNG with transparency
- Content: Foreground image only (Android will add background)

### Favicon (favicon.png)

- Size: 48x48 pixels
- Format: PNG

## Quick Start

For development, you can use placeholder images. Create simple images with your logo or text.
